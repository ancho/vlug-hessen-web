vlug-hessen

"vlug-hessen" steht für "Virtuelle Linux User Group Hessen". Die vLug-Hessen ist eine Linux User Group welche ausschließlich über Dienste im Internet kommuniziert. Für die Kommunikation und die Zusammenarbeit nutzen wir die folgenden Dienste:
  • Mailingliste
  • Git-Server
  • IRC
  • GnuSocial

Was ist unser Ziel?

Wir haben uns für dieses Projekt eigentlich zwei Ziele gesetzt.

  • Das erste Ziel ist die die Linux User auch über regionale Grenzen zusammen zu bringen und an Software und Dokumentationen für GNU/Linux zu arbeiten
  • Das zweite Ziel ist lokalen Linux User Groups in Hessen zu unterstützen

Auf der eine Seite sind wir der Meinung, dass die lokalen Gruppen nicht 'aussterben' sollten. Gerne posten wird auch Informationen zu Veranstaltungen etc. in unserem Blog.

Auf der anderen Seite wollen wir aber auch versuchen die Linux User zusammen zu bringen, um ggf. zusammen an Projekten etc. zu arbeiten (Software schreiben, Dokumentation erstellen). So wird beispielsweise ein IRC-Raum pro LUG sehr wenig besucht, aber ein IRC-Raum für die LUGs in Hessen könnte dann schon etwas voller werden. Über die vLUG können sich auch Gruppen bilden, welche dann über regionale Grenzen hinweg an Projekten arbeiten können.

Warum eine virtuelle LUG?

Nicht in allen Regionen in Hessen gibt es eine LUG. Des Weiteren gibt es verschiedene Gründe warum Personen leider nicht zu einem Stammtisch gehen können. Um auch für diese Personen eine "Platz" zu schaffen, haben wir die virtuelle LUG erstellt.

Wir würden uns auch freuen, wenn durch die vLUG zusammen an verschiedenen Projekten gearbeitet wird (Dokumentation, Programmierung, etc.), welches unser primäres Ziel ist. Gerne auch Personen die nicht aus Hessen kommen.

Unabhängig von den Angeboten die wir hier bereitstellen oder in Zukunft noch bereitstellen werden, besucht weiterhin die lokalen Stammtische!

Mitmachen

Du willst bei der vLUG-Hessen mitmachen? Dies geht ganz einfach! Sende eine kurze E-Mail mit folgenden Daten an kontakt@vlug-hessen.de.
  • Vor- und Nachname
  • E-Mail-Adresse
Wenn vorhanden, dann kann du uns auch gerne die folgenden Informationen zukommen lassen:
  • GPG-Key / Finterprint
  • github User

Mailing List

Dank tuxfamily.org haben wir jetzt zur Kommunikation eine Mailingliste. Du kannst dich mit einer E-Mail auf die Liste eintragen lassen. Sende eine E-Mail an "vlughessen-request $at$ lists $dot$ tuxfamily $dot$ org" mit dem Betreff "subscribe". Nachdem Sie die Anfrage bestätigt haben, wird diese noch mal von einem Admin von uns geprüft. Wir würde uns freunen, wenn du einfach auf der Liste dann mal "Hallo" sagst.
